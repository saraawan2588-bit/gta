How to play

  1. Use `Download archive` if you want the file directly from this page.
  2. Or use `Choose game.tar.gz` to select a file from your computer.
  3. Wait for the import to finish and confirm the local storage status above.
  4. Press `Start to play` and wait for the game data to load.
  5. You can import another local file later if you want to replace the current one.

Recommended: use a modern desktop browser for the smoothest setup and performance.
